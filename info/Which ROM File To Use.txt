
...........................Beakout Lava Lamp...........................

== Which ROM File to Use? ==

Most people will want to use: gb/breakout-lavalamp_mbc5.gb


== Game Boy ==
Standard MBC5 compatible ROM, use with a cart that has 8k (save) SRAM
- gb/breakout-lavalamp_mbc5.gb

Bare 32K cart ROM, no MBC and no (save) SRAM
- gb/breakout-lavalamp_nosave.gb

# For use with special Ferrante Crafts 31K ROM + 1K Save flash carts
# - gb/canyon_racer_*_31k_1kflash.gb


== Analogue Pocket ==
Standard MBC5 compatible ROM, use with a cart that has 8k (save) SRAM:
- pocket/breakout-lavalamp_mbc5.pocket

Bare 32K cart ROM, no MBC and no (save) SRAM:
- pocket/breakout-lavalamp_32k_nosave.pocket


== Mega Duck ==
Bare 32K cart such as the kind Inside Gadgets carries :
 - duck/breakout-lavalamp_32k_nosave.duck
